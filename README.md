# 装修报价微信小程序

## 项目简介
这是一个功能完整的微信小程序装修报价系统，支持多种装修项目的报价计算。

## 功能特点

### ✅ 核心功能
- **五大分类**：基础拆除类、墙面类、天花类、地面类、综合类
- **展开收缩**：每个大类都可以点击展开/收缩
- **多种输入**：支持面积、数量、米数、手动价格输入
- **智能计算**：实时计算总价，包含各种费用

### ✅ 特殊功能
- **垃圾外运计算**：4方车起步260元，每增加1平方65元
- **面议项目**：支持手动输入价格
- **费用明细**：设计费、运输费、管理费各2%
- **条件显示**：楼梯房上楼费可选显示

### ✅ 用户体验
- **默认全选**：进入页面默认勾选所有项目
- **实时计算**：输入即更新，无需手动刷新
- **界面美观**：现代化设计，响应式布局

## 安装使用

### 1. 克隆项目
```bash
git clone https://github.com/rocky6688/cxyj-applet.git
```

### 2. 导入微信开发者工具
- 打开微信开发者工具
- 选择"导入项目"
- 选择项目目录
- 填写AppID（可使用测试号）

### 3. 开始使用
- 点击"开始报价"进入报价页面
- 选择需要的装修项目
- 输入相应的面积、数量等参数
- 查看实时计算的总价
